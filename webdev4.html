    Deferred/Promise

    var defd = $.ajax({});
  defd.then(function(data){
    console.log(data);
  });
            
    var defd = new $.Deferred();
  defd.then(function(data){
    console.log(data); //-> 'hello';
  });
  defd.resolve('hello');
    • Nem csak jQuery létezik. pl: when.js
  • a deferred lényege a javascript erősségére építeni, a callback-ekre
  • gyakran nem a függvény visszatérési értéke számít, mert async a hívás
  • a deferred-del egy async hívás valódi eredményéhez tudunk további feladatokat csatolni

    • Single Page App

    Mi is ez?

    • Erre haladnak a fejlesztések. Sokszor nem is vagyunk tudatában, hogy használjuk
    • Lényege: klasszikus (szinkron) http lekérést csak az oldal első betöltésekor végzünk
    • Utána az aktuális oldal tartalmát frissítjük a kliens oldalon belülről (JavaScripttel)
    • Példa 1: Soundcloud: akármerre navigálok, a zene folyamatosan hallatszik
    • Példa 2: Facebook: chat
    • Példa 3: GMail
  • egyre inkább erre van szükség: egy alkalmazás fut a böngészőben

    • Single Page App

    Miben más a klasszikus oldaltól?

    • API-kon keresztül kommunikál a szerverrel, néha pjax
    • a szerver helyett a böngésző rendereli az oldalt
    • ergo egy teljes controller fut a böngészőben
    • nehézségek:
      deep linking
      megoldás: routing és state machine
      sok adat
      megoldás: model layer
      interaktivitás
      megoldás: two-way data binding
  • API Vs Session:
    • nincs session, nincs bejelentkezett felhasználó = stateless
    • HTML helyett JSON vagy JSONP
    • új nehézségek, pl: throttling

    Autocompletion Widget

    • ki szabad indulni a létező autocompletion kódból
    • a használata a pluginnak legyen a következő: 
      $('#searchInput').myComplete({
  target: $('#results'), 
  targetHandler: function(data){},
  ...
})
    • 3 karakter után induljon a keresés (állítható)
    • a keresőfüggvényt a beállítások közt lehessen definiálni vagy csak url-t megadni
    • az eredmények megjelenítését a targetHandler végzi
    • a plugin működését a last.fm API használatával mutasd is be
