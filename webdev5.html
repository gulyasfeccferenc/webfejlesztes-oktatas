AMD / CommonJS

Modulok JS-ben

  • A moduláris kód könnyebben karbantartható, átlátható
  • A modulok közt függőségek lehetnek
  • CommonJS: szabványok teljesértékű Js fejlesztéshez; ez ad egy Modul szabványt
  • AMD: Asynchronous Module Definition, kifejezetten böngészőre kitalálva
  • Részletek az AMD-ről és CommonJSről

Minták

CommonJS

var $ = require('jquery');
exports.myExample = function () {};

AMD

define(['jquery'] , function ($) {
    return function () {};
});

  • Állíts össze egy grunt konfigot a következő műveletekkel:
